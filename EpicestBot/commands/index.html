Built-in Commands

These are all of the built-in commands you can use with EpicestBot in any channel it is active in. Each built-in command has an associated required user-level. More information about user-levels can be found here.

Command Explanation User Level
?addcount [count] Adds a counter to yourself to track how many times you have said the given count. Everyone
?counts [username ?] Returns all counters for you or the given username. Everyone
?count [username ?] [count] Returns the number times you or the given username has used the given count. Counters must be initialized using ?addcount before being read. Everyone
?ecount [emote] Returns the number of times an emote has been used in chat. Everyone
?help Pings TheMostEpicest for help. Everyone
?toggle [on|off] Toggles the bot on or off in a channel. 'On' or 'Off" must be specified. Toggler
?addcount [username ?] [count] Adds a counter to yourself or the given username to track how many times you or they have said the given count. Counter
?delcount [username ?] [count] Deletes a counter from yourself or the given username. Counter
?reload Reloads all channel emotes and commands. Operator
?return [command] Returns the return for a given command. Operator
?property [command] [ul|cd|pre|case] Returns a given property for a given command. Operator
?prefix [newPrefix ?] Sets or returns the prefix of a channel. If no argument is passed, it will return the prefix, otherwise it will set the prefix to argument 1. (Deprecated) Operator
?version Returns to version number the bot is running. Operator
?addcom [command] [-[ul|cd|pre|case]=value ?] [return] Adds a new command. See Custom Commands for more information on custom commands. Operator
?editcom [command] [-[ul|cd|pre|case]=value ?] [return ?] Edits and existing command. See Custom Commands for more information on custom commands. Operator
?delcom [command] Deletes a given custom command. Operator
?togglesubmessages [on|off] Toggles sub messages on or off in a channel. 'On' or 'Off" must be specified. Operator
?togglecheermessages [on|off] Toggles cheer messages on or off in a channel. 'On' or 'Off" must be specified. Operator
?togglegiftmessages [on|off] Toggles gift messages on or off in a channel. 'On' or 'Off" must be specified. Operator
?togglecounters [on|off] Toggles counters on or off in a channel. 'On' or 'Off" must be specified. Operator
?addtoggler [username] Adds the role toggler to a given username. Operator
?removetoggler [username] Removes the role toggler from a given username. Operator
?addcounter [username] Adds the role counter to a given username. Operator
?removecounter [username] Removes the role toggler from a given username. Operator
?roles [username ?] Returns the roles for yourself or a given username. Operator
?operators Returns a list of operators for the channel. Operator
?counters Returns a list of counters for the channel. Operator
?togglers Returns a list of togglers for the channel. Operator
?addsubmessage [months] [message] Adds a sub message for a given number of months. Operator
?editsubmessage [months|default] [message] Edits a sub message for a given number of months or the default sub message. Operator
?delsubmessage [months] Deletes a sub message for a given number of months. Operator
?addcheermessage [bits] [message] Adds a cheer message for a given number of bits or more. Operator
?editcheermessage [bits|default] [message] Edits a cheer message for a given number of bits or more or the default cheer message. Operator
?delcheermessage [bits] Deletes a cheer message for a given number of bits or more. Operator
?addgiftmessage [gifts] [message] Adds a gift message for a given number of gift subs. Operator
?editgiftmessage [gifts|default] [message] Edits a gift message for a given number of gift subs or the default gift message. Operator
?delgiftmessage [gifts] Deletes a gift message for a given number of gift subs. Operator
?editdirectgiftmessage [message] Edits the direct gift message. Operator
?addoperator [username] Adds the role operator to a given username. Moderator
?removeoperator [username] Removes the role operator from a given username. Moderator
